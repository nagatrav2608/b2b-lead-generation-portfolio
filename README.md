# B2B Lead Generation & Data Cleaning Portfolio

## Project Overview
This repository showcases a high-quality, large-scale B2B lead generation and data hygiene project tailored for multinational sales and outbound marketing campaigns. The dataset focuses on C-level executives and key decision-makers within the Tech, SaaS, Logistics, and E-commerce industries across the United States (US) and the United Kingdom (UK).

The primary objective of this project was to extract raw contact lists, perform extensive data scrubbing, normalize text structures, and classify lead validity to minimize email bounce rates and optimize sales conversion metrics.

---

## Key Deliverables & Process
1. **Data Normalization & Formatting:** Cleaned raw scraped data into standard corporate structures using advanced formatting techniques (e.g., proper casing for names, lowercasing for emails, and standardized URL structures).
2. **Contact Segmentation:** Successfully separated full names into distinct `First Name` and `Last Name` columns to facilitate highly personalized email marketing sequences.
3. **Data Integrity & Hygiene:** Conducted rigorous deduplication checks and verified structural alignment for 50+ prospective target accounts.
4. **Lead Status Classification:** Implemented a systemized taxonomy to audit data accuracy, categorizing leads into `Verified`, `Catch-All`, and `Invalid`.

---

## Dataset Structure
The dataset consists of 10 structured columns optimized for modern CRM systems (such as HubSpot or Salesforce):

* **Company Name:** Standardized organization title.
* **Industry:** Core market vertical (SaaS, Fintech, Healthcare, etc.).
* **Website:** Cleaned root domain.
* **First Name & Last Name:** Separated prospect identity indicators.
* **Job Title:** Verified decision-maker corporate ranking (CEO, CTO, VP, Director).
* **Corporate Email:** Sanitized corporate contact address.
* **LinkedIn Profile:** Exact URL directory path for social selling.
* **Country:** Geographic market classification (US / UK).
* **Lead Status:** Quality control benchmark.
